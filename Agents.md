# PROJECT: Gym Landing Page

## Goal

Meningkatkan conversion user menjadi member gym dan mendapatkan traffic dari SEO.

## Target Market

Indonesia (Bahasa Indonesia, tone modern & simple)

## Design Direction

* Elegant
* Minimalist
* Premium
* Banyak whitespace
* Dark theme

## Tech Stack

* Next.js (App Router)
* TypeScript
* Tailwind CSS

## Architecture

* Feature-based
* Component-based
* No over-engineering

## Features

* Hero section
* Program
* Testimoni
* Pricing
* CTA
* WhatsApp button
* BMI calculator

## UX Principles

* Clear CTA
* Mobile-first
* Fast loading
* Simple navigation

## Performance Rules

* Use server components by default
* Use next/image
* Minimize JavaScript
* Lazy load heavy components

## SEO

* Bahasa Indonesia
* Target keyword: gym, fitness, personal trainer

## Avoid

* UI terlalu ramai
* terlalu banyak animasi
* arsitektur terlalu kompleks
