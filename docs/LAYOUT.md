# Landing Page Layout Structure

## Section Flow (Top to Bottom)

```
┌─────────────────────────────────────┐
│ 1. HERO SECTION                      │
├─────────────────────────────────────┤
│ 2. VALUE PROPOSITION                 │
├─────────────────────────────────────┤
│ 3. PROGRAMS                          │
├─────────────────────────────────────┤
│ 4. TRANSFORMATION                    │
├─────────────────────────────────────┤
│ 5. TESTIMONIALS                      │
├─────────────────────────────────────┤
│ 6. PRICING                           │
├─────────────────────────────────────┤
│ 7. FINAL CTA                         │
└─────────────────────────────────────┘
```

---

## 1. Hero Section

**Purpose:** Grab attention, communicate value immediately

**Content:**
- Headline: Bold, benefit-driven (e.g., "Transform Your Body in 90 Days")
- Subheadline: Support the main claim
- Primary CTA: "Mulai Sekarang" (scrolls to pricing)
- Secondary CTA: "Lihat Program" (scrolls to programs)
- Background: Dark, premium image or gradient

**Conversion Goal:** Get visitor to take action immediately

---

## 2. Value Proposition

**Purpose:** Build trust, explain why choose us

**Content:**
- 3-4 key benefits (not features)
- Icons + short text
- Focus on results, convenience, expertise

**Example:**
- Personal Trainer Bersertifikat
- Program Terukur & Personalized
- Fasilitas Premium & Bersih

**Conversion Goal:** Establish credibility before asking for commitment

---

## 3. Programs

**Purpose:** Show what's available, help visitor choose

**Content:**
- 3-4 program cards
- Each card: title, description, benefits, price/hint
- Programs: Weight Loss, Muscle Building, Personal Training, etc.

**Conversion Goal:** Help visitor identify which program fits their goal

---

## 4. Transformation

**Purpose:** Social proof through visual results

**Content:**
- Before/after images (real results)
- Brief story: name, time frame, result
- Minimal text, focus on visuals

**Conversion Goal:** Show real results are achievable

---

## 5. Testimonials

**Purpose:** Build trust through social proof

**Content:**
- 3-4 testimonials
- Photo + name + quote
- Authentic, specific results

**Conversion Goal:** Remove doubt, show others succeeded

---

## 6. Pricing

**Purpose:** Clear pricing, make decision easy

**Content:**
- 3 pricing tiers
- Highlight "Most Popular" option
- Clear what's included
- CTA button per plan

**Conversion Goal:** Close the sale

---

## 7. Final CTA

**Purpose:** Last push for undecided visitors

**Content:**
- Simple, urgent message
- Single strong CTA
- WhatsApp contact option

**Conversion Goal:** Capture fence-sitters

---

## Conversion Flow Logic

1. **Hook** → Hero grabs attention
2. **Trust** → Value proposition establishes credibility
3. **Interest** → Programs show options
4. **Belief** → Transformation/testimonials prove results
5. **Decision** → Pricing makes choice clear
6. **Action** → Final CTA seals the deal

## Design Principles Applied

- **Mobile-first:** Each section stacks vertically
- **Whitespace:** Generous spacing between sections
- **Dark theme:** Premium feel throughout
- **Minimal text:** Scannable, not overwhelming
- **Clear CTAs:** Always visible what to do next

## WhatsApp Floating Button

- Fixed position (bottom-right)
- Always visible
- Direct to WhatsApp chat
- Alternative conversion path
